# Fibonacci Iterative

## Algorithm name

Fibonacci Sequence (Iterative)

## Author

Victor Manuel Montini (2026-03-23)

## Algorithm description

The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones, starting from 0 and 1. The iterative approach computes the sequence by maintaining a running list and repeatedly appending the sum of the last two elements until the desired number of terms is reached. This avoids the exponential time cost of the naive recursive approach.
