# BFS

## Algorithm name

Breadth-First Search (BFS)

## Author

Victor Manuel Montini (2026-03-23)

## Algorithm description

Breadth-First Search is a graph traversal algorithm that explores all neighbors of a node before moving to nodes at the next depth level. It uses a queue to keep track of nodes to visit next. BFS is useful for finding the shortest path in unweighted graphs, checking connectivity, and level-order traversal. Time complexity is O(V + E) where V is vertices and E is edges.
